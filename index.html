LazyDeveloperHelper

Too lazy for "pip install"? Manage Python, Rust, Lua, C/CPP, Ruby, (and etc more) and Gradle dependencies effortlessly in Neovim.

Install Now

Key Features

  • Automatic dependency installation based on file type
  • No need configuration and setup
  • Python dependencies via pip
  • Rust crates via Cargo
  • Kotlin/Groovy via Gradle
  • Seamless Neovim integration
  • Support for LuaRocks
  • Support for RubyGems
  • Support for C/C++ via vcpkg (not completely), Conan, Nuget
